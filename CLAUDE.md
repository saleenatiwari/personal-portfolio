## Blender file naming (always follow)
When saving a new Blender file, always save as the next version number — e.g. if the last file is `paint_v3.blend`, save new work as `paint_v4.blend`. Never overwrite an existing `.blend` file. (Blender's auto-backup creates a `.blend1`, which is not a real working file and loses version history.)
